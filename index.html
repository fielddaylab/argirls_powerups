Photo Editing 101 (Apple Photos)

This powerup is part of the following badges:

Media GuruDigital Artist

The purpose of this Power Up is to learn how to pull files down from google drive, do some basic editing with Apple Photos, then save them so they can be used by other digital media software.

Media Workflow

This powerup is part of the following badges:

Media Guru

In this Power Up you will use Airdrop and Google drive to move your media between devices and people.

Media Guru
